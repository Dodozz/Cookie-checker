Due to browser security (CORS), choose one option to connect:

🟢

Option 1: Local Proxy

RECOMMENDED - Most reliable method. Runs a local server to bypass CORS.

  1. Download proxy-server.zip
  2. Extract files to a folder
  3. Install Node.js from nodejs.org
  4. Open terminal in folder
  5. Run: npm install
  6. Run: npm start
🔌

Option 2: CORS Extension

Quick setup - Install a browser extension to allow cross-origin requests.

  • Install: Allow CORS Extension
  • Enable the extension
  • Refresh this page
  • Try connecting below

⚠️ May not work reliably with authentication cookies

💻

Option 3: Console Script

Run directly from Roblox.com console - No CORS issues.

  1. Go to roblox.com (logged in)
  2. Press F12 → Console tab
  3. Copy script below
  4. Paste and press Enter
