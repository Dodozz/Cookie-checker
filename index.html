⚠️ Nota Importante sobre CORS

Debido a las políticas de seguridad CORS de Roblox, esta aplicación no puede hacer peticiones directas a sus APIs desde el navegador. Para usar esta herramienta, necesitarás:

  • Usar una extensión de navegador para deshabilitar CORS temporalmente (ej: CORS Unblock)
  • O usar un proxy CORS (ej: https://corsproxy.io/?)
  • O ejecutar esta página desde un servidor con proxy backend
